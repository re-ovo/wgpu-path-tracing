# wgpu-path-tracing

A path tracing implementation using WebGPU.

## Usage

```bash
npm install
npm run dev
```

## Features

- [ ]
- [ ] GLTF Scene Loading
- [ ] BVH Acceleration
- [ ] Multiple Importance Sampling (MIS)

## GLTF Extensions Supported

- KHR_materials_ior
- KHR_materials_transmission
- KHR_materials_emissive_strength

## Libraries

- TypeGPU
- @loaders.gl/gltf
